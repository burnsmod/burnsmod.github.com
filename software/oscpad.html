Features

OSCPad is an OSC controller for Android, featuring:

  • Use custom TouchOSC layouts on Android.
  • 3D accelerated UI for lowest latency.
  • Custom OSC layouts can be loaded and controlled.
  • Create custom palette for maximum control of look and feel.
  • Fast and responsive multi-touch interface optimized for low-light situations.
