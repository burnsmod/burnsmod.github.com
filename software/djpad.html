Features

DJPad is an MP3 audio mixing application for most mobile OSes, featuring:

  • 3D accelerated UI for lowest latency.
  • Parallel waveforms to visually aid in mixing
  • Dual players featuring cuepoints and turntable scratching.
  • Support for more audio formats including MP3, Apple iTunes AAC, OGG, FLAC, and WAV.
  • Hi-fidelity mixer with 3-band EQ.
  • Mono split-output option allowing true separate Cue & Master channels for live mixing
  • Fast and responsive multi-touch interface optimized for low-light situations.

Future

As well, DJPad is actively under development. Features that are coming soon for all platforms:

  • FX
  • ID3 tag loading (including BPM tags if present)
  • Playlists & auto-play functionality
